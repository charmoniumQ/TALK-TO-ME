# TALK TO ME!!
A chatbot

1. Clone the repo.
2. Download your data according to [`talk_to_me/fb_data2exchanges.py:L13`]
3. Modify `main.py` to point to your data and your name.
4. Run `main.sh` (It will take care of venv, requirements, etc.).
